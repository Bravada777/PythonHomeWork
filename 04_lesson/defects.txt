1. БАГ 001
   Название: Неправильная обработка строки с пробелами в начале в функции `capitalize`
   Описание: Функция `capitalize` не удаляет пробелы в начале строки перед тем, как сделать первую букву заглавной.
   Шаги для воспроизведения:
   1. Вызвать функцию `capitalize` с аргументом `"  hello"`.
   Ожидаемый результат: `"Hello"`.
   Реальный результат: `"  hello"`.

2. БАГ 002
   Название: Неправильный результат функции `contains` при передаче пустой строки
   Описание: Функция `contains` возвращает `True` при проверке пустой строки на наличие пустого символа, хотя ожидаемый результат — `False`.
   Шаги для воспроизведения:
   1. Вызвать функцию `contains` с аргументами `("", "")`.
   Ожидаемый результат: `False`.
   Реальный результат: `True`.

3. БАГ 003
   Название: Некорректное удаление символа в функции `delete_symbol`
   Описание: Функция `delete_symbol` оставляет лишний символ при удалении последнего вхождения символа.
   Шаги для воспроизведения:
   1. Вызвать функцию `delete_symbol` с аргументами `("Hello World", "l")`.
   Ожидаемый результат: `"Heo Wor"`.
   Реальный результат: `"Heo Word"`.

4. БАГ 004
   Название: Некорректная обработка `None` в функциях класса `StringUtils`
   Описание: Функции класса `StringUtils` выдают ошибку `AttributeError` при передаче значения `None`, хотя ожидается исключение `TypeError`.
   Шаги для воспроизведения:
   1. Вызвать любую функцию класса `StringUtils` с аргументом `None`.
   Ожидаемый результат: Исключение `TypeError`.
   Реальный результат: Исключение `AttributeError`.
